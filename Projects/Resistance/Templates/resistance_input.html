Will my antibiotic still work?

This tool analyses mutations in the beta-lactamase gene which can confer resistance to beta-lactam antibiotics (such as penecillin) to bacterial which express it.

In the box below you can enter a DNA mutation in the beta lactamase coding sequence and the program will work out whether this will cause a change in the protein produced. If the protein changes then the program will say whether it is an amino acid substitution, or whether the introduction of a premature stop codon would cause the entire protein to be truncated.

The form of the mutation string must be:

[original base][base position][new base]

eg G3A would be a change of G in position 3 to be A.
